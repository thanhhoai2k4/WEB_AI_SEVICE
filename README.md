# Hướng dẫn cài đăt:


## clone github

vao cmd và thực hiện:
```
    git clone https://github.com/thanhhoai2k4/WEB_AI_SEVICE.git
```


## thực hiện cài đặt thư viện:

```commandline
cd WEB_AI_SEVICE
bash bin/set_up_venv.sh
```


## chay server

```commandline
 python .\manage.py runserver
```
